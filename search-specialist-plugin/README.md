# Search Specialist Plugin

A comprehensive search specialist plugin for advanced information retrieval, query optimization, and knowledge discovery across diverse sources.

## Overview

The Search Specialist plugin provides an expert agent that excels at finding precise, relevant information efficiently. It specializes in needle-in-haystack searches, comprehensive research, and high-quality result curation across multiple domains and source types.

## Features

- **Advanced Query Optimization**: Boolean operators, proximity searches, wildcard usage, and semantic search
- **Multi-Source Expertise**: Web search engines, academic databases, patent databases, legal repositories, government sources
- **Comprehensive Coverage**: Systematic searching with precision targeting and recall optimization
- **Quality Assessment**: Source credibility evaluation, authority verification, and bias detection
- **Efficient Workflow**: Search automation, batch processing, and result caching

## Components

### Agent: Search Specialist

The core agent that provides advanced search capabilities:

- **Proactive Triggering**: Automatically activates when complex search needs are detected
- **Comprehensive Tools**: Uses Read, Grep, Glob, WebFetch, and WebSearch for thorough research
- **Domain Expertise**: Specialized knowledge across scientific literature, technical documentation, legal research, and more
- **Quality Metrics**: Maintains >90% precision rate with comprehensive coverage reporting

### Command: /search-specialist

A concise command to launch the search specialist agent with a specific query:

```bash
/search-specialist fixing aur package errors on omarchy
/search-specialist comprehensive research on quantum computing applications
/search-specialist find recent developments in renewable energy storage
```

## Installation

1. Copy the plugin to your Claude Code plugins directory
2. The agent will be available for proactive search assistance
3. Use the `/search-specialist` command for direct search requests

## Usage Examples

### Proactive Usage
The agent will automatically trigger when it detects complex search needs:

- "I need comprehensive information about..."
- "Find detailed research on..."
- "Search for authoritative sources about..."

### Command Usage
Directly invoke the search specialist:

```bash
/search-specialist latest developments in AI safety research
/search-specialist technical documentation for Kubernetes networking
/search-specialist legal precedents for data privacy cases
```
